Pseudo-Voigt function

$$y = y_0 + A\left\{\mu\,\frac{2}{\pi}\,\frac{w}{4\,(x-x_c)^2+w^2} + \left(1-\mu\right)\,\frac{\sqrt{4\,\ln{2}}}{\sqrt{\pi}\,w}\,\exp\left[-\frac{4\,\ln{2}}{w^2}\,(x-x_c)^2\right]\right\}$$

Gaussian profile

$$y = y_0 + A\,\frac{\sqrt{4\,\ln{2}}}{\sqrt{\pi}\,w}\,\exp\left[-\frac{4\,\ln{2}}{w^2}\,(x-x_c)^2\right]$$

Lorentz function

$$y = y_0 + A\,\,\frac{2}{\pi}\,\frac{w}{4\,(x-x_c)^2+w^2}$$
